padding 예제

width : 300px
height : 300px
box-sizing : content-box
padding : 0px
*실제 요소의 너비는 300px(컨텐츠)
+ 1px(왼쪽테두리) + 1px(오른쪽 태두리)
=302px
width : 300px
height : 300px
box-sizing : content-box
padding : 20px
content-box에서 width는 컨텐츠 너비 *실제 요소의 너비는 300px(컨텐츠)
+ 20px(왼쪽패딩) + 20px(오른쪽패딩)
+ 1px(왼쪽테두리) + 1px(오른쪽 태두리)
=342px
width : 300px
height : 300px
box-sizing : border-box
padding : 20px
border-box에서 width는 컨텐츠 + 패딩 + 테두리 너비
*실제 컨텐츠의 너비는 300px(실제 요소의 너비)
- 20px(왼쪽패딩) - 20px(오른쪽패딩)
- 1px(왼쪽테두리) - 1px(오른쪽 태두리)
=258px(컨텐츠)

슈퍼키
- 한행을 찾을수 있는 속성들의 집합

후보키
- 슈퍼키들 중에서 속성들이 최소인 집합

기본키
- 후보키 중 선정된 키
- NULL값을 가질 수 없다
- 기본키로 검색하면 테이블에서 오직 하나의 행만 존재한디(최대)

대체키
- 기본키로 선정되지 않은 후보키

복합키
- 키가 여러 속성으로 이루어진 경우

대리키
- 1,2,3,와 같이 편하게 쓸수 있는 기본키
- 순차적으로 증가하는 기능을 제공
- 기본키가 복합키인 경우

DDL
- create : 추가
- drop : 삭제
- alert : 수정

DML
- insert : 데이터 추가
  - insert values
  - insert select
- update : 데이터 수정
  - set
- delete : 데이터 삭제
- select : 데이터 조회
select * from
where
   조건식
order by 컬럼1 방법
limit 번지, 개수;

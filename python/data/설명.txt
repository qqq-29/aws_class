CNN으로 패션 이미지 학습한 내용을 FastAPI에 적용

1. 모듈을 생성(fashion.py)
 - 모델을 저장
 - 모델을 불러오는 기능
 - 예측하는 기능
2. 스프링 부트
 - 화면 추가
 - 이미지 업로드
 - 서버에서는 업로드된이미지를 AI서버에게 전달
3. AI서버
 - 이미지를 받아서 fashion을 이용하여 예측을 결과를 반환
 
 
머신러닝에서 이미지 학습
- 2차원 이미지를 1차원으로 나열해서 학습
	=> 점들 사이의 관계를 모름
	
딥러닝에서 이미지를 학습
- 2차원 이미지를 그대로 학습(CNN)
	=> 점들 사이의 관계를 학습
- 학습 과정
	=> 필터를 통해 특징을 추출(3x3)
	=> 압축(이미지 크기가 줄어듬)
	
머신러닝에서 텍스트 학습
- 단어 개수
- 학습 과정
	=> 형태소 분리(불용어-감탄사등과 같이 학습에 필요없는 단어)
		- 가,이,아와 같은 조사나 감탄사는 개수셀때 의미가 없음
		- 갔다, 갔었다,가니를 가다로 함게 세기 위해
	=> 분리된 단어들을 다시 하나의 문장
	=> 벡터화(숫자로 변경)
		- 각 단어가 몇개 있는지를 숫자로 변경
		- 문자열 리스트 => 숫자가 들어있는 행렬로 변경
딥러닝에서 텍스트 학습
	=> 형태소 분리(불용어 + 기본형 - 감탄사등과 같이 학습에 필요없는 단어)
		- 가,이,아와 같은 조사나 감탄사는 개수셀때 의미가 없음
		- 갔다, 갔었다,가니를 가다로 함게 세기 위해
	=> 분리된 단어들을 다시 하나의 문장
	=> 벡터화(숫자로 변경)
		- 단어가 사이의 관계를 표현(128개의 차원에 표현)
